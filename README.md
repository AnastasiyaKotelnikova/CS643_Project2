# CS643 Project 2 – Parallel Training & Inference on AWS EMR

This project demonstrates distributed model training using Apache Spark on AWS EMR, and model inference using Docker and AWS EC2.

Folder Structure:
- `/training/` – PySpark code for EMR training
- `/inference/` – model inference scripts
- `/docker/` – container setup
- `/docs/` – screenshots and project report

Output:
- Trained model stored in S3
- Docker container on Docker Hub
- GitHub repo with code and documentation
